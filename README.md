# 그누보드 페이지 게시판

페이지 게시판 스킨은 그누보드의 게시판을 페이지 형태로 사용하기 위해 개발된 스킨입니다.
그누보드 게시판 스킨이며 에디터(CKEditor)로 페이지를 수정하는 것과 같은 형태의 기능을 제공합니다.
